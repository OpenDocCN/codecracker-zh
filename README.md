## 下载

### Docker

```
docker pull apachecn0/codecracker-zh
docker run -tid -p <port>:80 apachecn0/codecracker-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g codecracker-zh
codecracker-zh <port>
# 访问 http://localhost:{port} 查看文档
```
