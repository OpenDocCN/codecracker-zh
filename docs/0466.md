## CSS 背景-附件语法

CSS 中**背景附件**属性的语法是:

```
background-attachment: x;
```

**x** 的值应该是下列值之一:

*   **固定** -用于固定背景图像
*   **滚动** -用于随页面/内容一起滚动背景图像。这是默认值
*   **local** -用于随元素内容一起滚动背景图像
*   **初始** -用来使用默认值
*   **inherit** -用于使用父元素继承的值
