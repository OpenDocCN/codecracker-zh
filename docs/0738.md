# JavaScript 对话框

> 原文：<https://codescracker.com/js/js-dialog-box.htm>

JavaScript 对话框有以下三种类型:

*   用于确认任何用户输入的 JavaScript 对话框
*   用于引发警告框的 JavaScript 对话框
*   JavaScript 对话框，用于用户的一种输入

*   [JavaScript 确认框](/js/js-confirm-box.htm)
