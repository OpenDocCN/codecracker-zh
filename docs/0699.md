## CSS 标题侧语法

CSS 中**标题侧**属性的语法是:

```
caption-side: x;
```

**x** 的值应该是下列值之一:

*   **顶部** -将标题移动到表格顶部。这是<u>默认的</u>值
*   **底部** -将标题移动到表格底部
*   **初始** -用来使用默认值
*   **inherit** -用于使用父元素继承的值
