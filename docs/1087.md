# Java 编号

在 Java 中，你可以自由地处理数字，使用基本的数据类型，如 byte、int、double、long 等。下面是一些在 Java 中定义和初始化变量的例子:

```
int numi = 432;
float numf = 43.23;
```

## Java 数字示例

下面是一个例子，在 Java 中使用数字:

```
/* Java Program Example - Java Numbers */

class JavaProgram
{
    public static void main(String args[])
    {

        Integer num = 50;
        num =  num + 100;
        System.out.println(num); 

    }
}
```
