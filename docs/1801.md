# C++程序将文件中每个单词的首字母大写

> 原文：<https://codescracker.com/cpp/program/cpp-capitalize-every-word-in-file.htm>

## 文件中的所有单词都要大写

问题是，**用 C++写一个程序，在运行时从用户那里接收文件名，并大写该文件中可用的每个单词 。**下面是它的回答:

```
#include<iostream>
#include<fstream>
#include<sstream>

using namespace std;
int main()
{
   ifstream inf;
   ofstream ouf;
   stringstream str_stream;
   string str;
   char filename[30];
   int i=0, temp=0;

   cout<<"Enter File's Name: ";
   cin>>filename;

   inf.open(filename);
   str_stream<<inf.rdbuf();
   inf.close();
   str = str_stream.str();
   while(str[i])
   {
      if(temp==0)
      {
         str[i] = toupper(str[i]);
         temp = 1;
      }
      else if(isspace(str[i]))
         temp = 0;
      i++;
   }
   ouf.open(filename);
   ouf<<str;
   cout<<"\nAll words capitalized Successfully!";
   cout<<endl;
   return 0;
}
```
