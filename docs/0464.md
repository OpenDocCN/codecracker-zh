## CSS 背景来源语法

CSS 中 **background-origin** 属性的语法是:

```
background-origin: x;
```

**x** 的值应该是下列值之一:

*   **边框** -用于将背景图像的原点位置设置到左上角
*   **填充框** -用于设置**背景图**的原点位置到左上角填充 。这是默认值。
*   **内容框** -用于将**背景图**的原点位置设置到内容 的左上角
*   **初始** -用来使用默认值
*   **继承** -用于使用父元素继承的值
